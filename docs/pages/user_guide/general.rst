GCMotion User Guide
===================

This guide is an overview of the GCMotion packages and goes through the most important 
features using examples. Details are found in the :ref:`API Reference <apiref>`

.. rubric:: Importing the package
    :heading-level: 3

To use the package we of course have to import it first. Using the abbreviation *gcm*
always comes in handy. Importing numpy is also useful for defining the initial condtions
easier:

.. code-block:: python

    >>> import gcmotion as gcm
    >>> import numpy as np
